# DriveStats

iOS app that tracks real-life driving data (top speed, distance, time in car) and ranks friends on private group leaderboards.

**Constraints:** $0 budget · iOS only · sideloaded via [AltStore](https://altstore.io) with a free Apple ID · developed on Windows (Mac later), so all iOS builds happen in GitHub Actions.

## Stack

| Layer | Choice |
|---|---|
| UI | SwiftUI, min iOS 17 |
| Tracking | CoreLocation + CoreMotion (automatic drive detection) |
| Pure logic | `Packages/DriveStatsCore` — Foundation-only SwiftPM package, unit-tested on Linux CI |
| Backend | Supabase free tier (Auth, Postgres + RLS, Realtime) — added in Milestone 1 |
| Project file | Generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml` (never edit `.xcodeproj` by hand) |

## Build & install pipeline

1. Push to `main` → GitHub Actions runs `core-tests` (Linux) then `build-ipa` (macOS runner):
   XcodeGen generates the project, `xcodebuild` builds **unsigned**, output zipped as `DriveStats.ipa` artifact.
2. Download the `DriveStats-ipa` artifact from the Actions run and unzip it.
3. On the Windows PC: install AltServer (requires iTunes + iCloud, the non–Microsoft Store versions), install AltStore to the iPhone, then sideload `DriveStats.ipa` — AltStore signs it with your free Apple ID and refreshes the 7-day profile automatically over Wi-Fi.

Free Apple ID limits: max 3 sideloaded apps, profiles expire after 7 days (AltStore auto-refreshes), no push notifications / iCloud / Sign in with Apple — the architecture avoids all of these.

## Local build (once a Mac is available)

```sh
brew install xcodegen
xcodegen generate
open DriveStats.xcodeproj
```
